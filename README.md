# Mimoora Landing Page

Landing page for the Mimoora kids surprise bath bomb product.

## Files

- `index.html`: main landing page
- `styles.css`: page styles
- `assets/`: selected product and lifestyle images

## Notes

- WhatsApp ordering is wired in `index.html`
- Shipping policy is configured for Greater Cairo and other governorates
- Prices are currently set to:
  - `999 EGP` for the 6-piece box
  - `1499 EGP` for the 12-piece box
